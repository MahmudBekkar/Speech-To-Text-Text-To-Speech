# 🎤 Speech-to-Text & Text-to-Speech

## 📌 Description

This project is a simple C# Windows Forms application that converts:

* 🗣️ Speech ➝ Text
* ✍️ Text ➝ Speech

It allows users to interact with the system using voice and text in an easy way.

---

## 🚀 Features

* 🎤 Speech Recognition (Speech to Text)
* 🔊 Voice Output (Text to Speech)
* ▶️ Start / Stop Listening
* 🧹 Clear Text
* ❌ Exit Application

---

## 🛠️ Technologies Used

* C#
* Windows Forms
* System.Speech

---

## ▶️ How to Run

1. Open the project in Visual Studio
2. Build the solution
3. Run the program (F5)
4. Click **Start Listening** and speak
5. Type text and click **Speak**
